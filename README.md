# Análise Comparativa de Redes Escolares

A análise foi realizada em 3 escolas que já possuem conexão via fibra óptica da RNP e de um provedor local. As redes foram testadas separadamente, sendo realizado um teste conectado ao equipamento da RNP e outro ao equipamento da rede local.

As pastas organizadas por escola contêm gráficos comparando o desempenho das duas redes e outras métricas relevantes. Algumas métricas que não aparecem nos gráficos visuais (como o TTL - Time To Live) também foram coletadas e serão detalhadas no trabalho escrito.
